ls-> List of all the files and directories in that folder
ls -l -> List in Long format
ls -a -> list of all files and dir. including hidden ones also
pwd -> present working directory
cp source destination -> copy files
cp -v sourcce destination -> copy files and tells what is copied where.(v- verbose)
mv source destination -> Moves the files (just like cut and paste).
mv -v source destination -> moves and tells which is moved where (v - verbose)
mkdir -> Creates a directory.
touch -> Creates a File.
rmdir -> Deletes an empty directory.
rm -r dir -> Deletes a non-empty directory.
cat file.txt -> Shows all the contents of the file on the terminal.
cat > file.txt -> if old file then overwrites the file and you can start writing from the beggining else new file is created.
cat file1.txt > file2.txt -> Copies the content of file1 in file 2.
grep "hello" file.txt -> Prints all the lines that have "hello" in them.
grep -i "hello" file.txt -> Case insensitive.
grep "^H" file.txt ->  Start with H.
grep "x$" file.txt -> End with x.
grep "^l.*x$" file.txt -> line starting with l and ending with x.
grep [hf] file.txt -> The Line either contains "h" or "f".
